フォルダの名前をプロジェクト内で統一したい！
機能フォルダが直感的に判別できたら作業効率があがるのに！
そんな悩みを簡単に解決できるアセットです。

どうやって使うの？:
1, 普段通りにフォルダを作ります。
2, "Assets/Editor Default Resources/CustomFolderIcons"の中のアイコンと同じ名前なら、そのアイコンに変わります。
* 1文字でも違うとそのアイコンになりません。
3, 見事、フォルダ名を正しく付けられました！ (もし名前が正しくないとアイコンが変わらないのですぐに気づきますよね？)


他にも統一したいフォルダ名がある！
アイコン画像を変更したい！

アイコン画像の変更/追加:
1,"Assets/Editor Default Resources/CustomFolderIcons"フォルダに入ります。
2,ここのアイコンの名前を変えたり、アイコンを追加したりします。
* アイコン画像の Import Setting:
　Texture Type: "Cursor"
　Filter Mode: "Point (no filter)"
3, 見事、新しくフォルダ名の規則を追加できました！
